Ups for Commerce
------------------------

UPS shipping method for fetching rates via the UPS API. Setup your UPS developer account at https://www.ups.com/upsdeveloperkit to use this module.

After making a UPS developer account, you must generate an API key and enter your API key, UPS user ID, and UPS password into the system settings before using this module. Afterwards, you can add shipping methods in the Commerce configuration.

Note that not all UPS shipping methods may be available for your account. Currently it is pulling in all possible UPS shipping methods.
